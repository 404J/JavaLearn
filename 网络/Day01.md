# Day01

## 网络层级

- 应用层：包含 HTTP 协议，组装数据，用户应用程序

- 传输控制层：TCP/IP，UDP 协议
  TCP建立连接，3次握手。3次保证客户端和服务端都确认可以建立连接（output 都有 input 回应）
  建立连接后进行数据传输，
  断开连接，4次分手，保证连接可靠性。
  
  ```shell
  netstat -natp # 查看当前所有的 tcp 连接
  ```

- 网络层
  每个网络中的机器存储自己周边的路由表，根据路由表匹配到下一个节点

- 链路层
  根据网络层的下一个节点匹配到 MAC 地址

- 物理层

> 浏览器属于应用层，其他层次在操作传统内核，
